# Security Policy

## Supported Versions

We currently support the following versions of Kaelith Nova Portfolio:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you find a security vulnerability, please do not open a public issue. Instead, please report it privately to the maintainer.

We take security seriously and will respond to reports within 48 hours.
